# whu-thesis 修改记录

本文档格式遵守[如何维护更新记录](https://keepachangelog.com/zh-CN/1.0.0/)。

## Unreleased

## v2.4.0 2025-03-17

### Added

+ 当 `subject` 为空时只显示 `major`
+ 在 `demo.tex` 中新增了部分使用注释
+ 新增 `achievement.tex` 用于添加科研成果
+ 增加对 lualatex 的支持

### Deleted

+ 移除对 xeCJKfntef 的依赖

## v2.3.0 2025-01-07

### Fixed

+ 修复中文摘要页眉页脚问题 (#288)
+ 修复英文封面日期错误问题 (#302, #303)
+ 修复标题出现换行控制符时页眉标题也出现换行的问题 (#304, #305)

### Added

+ 新增布尔键 `chapter-page-header` 控制是否在每一章首页输出页眉

## v2.2.0 2024-05-29

### Added

+ 新增布尔型键 `license` 用于控制是否输出学位论文使用授权协议书

### Removed

+ 移除博士论文中使用授权协议书的输出部分

## v2.1.1 2024-05-22

### Fixed

+ 修复模块文件夹未打包的问题

### Added

+ 增加在线编译平台中文字体调用的说明
+ 增加对部分字体的注解

### Removed

+ 移除 `.cls` 文件中处理计院/网安英文封面的历史遗留代码
+ 移除手册中对 QED symbol 的说明

## v2.1.0 2024-05-18
